**2. Rewrite `SaveManager.cs`** — Replace the generic `Save<T>/Load<T>` API with a unified `GameSaveData`-specific API. Keep the class signature `public class SaveManager : MonoSingleton<SaveManager>`. Key changes:

- **Field:** `private GameSaveData _currentData;` — in-memory cache of the save data
- **Field:** `private const string SaveKey = "game";` — single save file name
- **OnInitialize():** Same directory setup (`Path.Combine(Application.persistentDataPath, "Saves")`), then call `LoadFromDisk()` to auto-load existing save (per FOUND-05). If no save exists, create via `GameSaveData.CreateNew()`.
- **Public API:**
  - `public GameSaveData Data => _currentData;` — read access to current save
  - `public void Save()` — serializes `_currentData` via `JsonUtility.ToJson`, calls `WriteAtomic(json)`, updates `_currentData.lastSaveTimestamp`
  - `public void LoadFromDisk()` — attempts to read main file; if deserialization fails or file corrupt, tries `.bak` file; if both fail, creates new via `GameSaveData.CreateNew()`. Calls `MigrateIfNeeded()` after successful load.
  - `public bool Exists()` — checks if save file exists on disk
  - `public void DeleteAll()` — deletes `.json`, `.tmp`, and `.bak` files, resets `_currentData` to `GameSaveData.CreateNew()`
- **Private methods:**
  - `private void WriteAtomic(string json)` — writes to `{path}.tmp`, then if `{path}` exists calls `File.Replace(tmpPath, finalPath, bakPath)`, else calls `File.Move(tmpPath, finalPath)`. Wrapped in try/catch logging errors.
  - `private void MigrateIfNeeded()` — if `_currentData.saveVersion < GameSaveData.CreateNew().saveVersion`, call `_currentData.MigrateFrom(_currentData.saveVersion)` and save.
  - `private string GetPath()` — returns `Path.Combine(_saveDir, "game.json")`
- **Auto-save trigger (per D-04):** Add `OnApplicationFocus(bool hasFocus)` — when `hasFocus == false`, call `Save()`. Other auto-save triggers (habit completion, resource update, report submit) will be wired by consuming systems in later phases via `SaveManager.Instance.Save()`.
- **Crash recovery:** In `LoadFromDisk()`, if `File.ReadAllText` or `JsonUtility.FromJson` throws on the main file, log a warning, check for `.bak` file, attempt to load that instead.
